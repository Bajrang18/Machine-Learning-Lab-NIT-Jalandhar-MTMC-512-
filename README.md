## Repository Overview
This repository contains **all laboratory assignments** completed as part of the **Machine Learning Lab (MTMC-512)**.  
The lab work is designed to provide **hands-on exposure** to the theoretical concepts of Machine Learning through **Python-based implementations**.

The assignments gradually progress from **numerical computing and data preprocessing** to **core machine learning algorithms**, aligning strictly with the official syllabus.

---

## Course Objectives
- To provide practical exposure to Machine Learning constructs
- To enable students to solve real-world problems using ML techniques
- To develop understanding of probabilistic modeling and learning algorithms
- To gain experience with supervised, semi-supervised, and unsupervised learning methods

---

## Lab Assignment Structure (Syllabus-Aligned)

### 🔹 Assignment 1: Python, NumPy, and Pandas Foundations
- NumPy array creation, reshaping, slicing, and broadcasting
- Vectorized operations and matrix computations
- Statistical measures (mean, median, standard deviation, Z-score)
- DataFrame operations and indexing (`loc`, `iloc`)
- Handling missing values and duplicates
- Data transformation (wide to long format)
- Rolling statistics and basic time-series analysis

> **Focus:** Data handling and numerical foundations for Machine Learning

---

### 🔹 Assignment 2: Probability and Statistical Modeling
- Probability distributions (Normal, Binomial, Exponential, etc.)
- Sampling techniques and empirical analysis
- Central Limit Theorem demonstrations
- Covariance and correlation analysis
- Bias–variance intuition through simulations

> **Focus:** Probabilistic foundations of Machine Learning

---

### 🔹 Assignment 3: Linear and Logistic Regression
- Simple and multiple linear regression
- Cost functions and gradient descent
- Logistic regression for binary classification
- Model evaluation metrics (MSE, accuracy)
- Overfitting and generalization analysis

> **Focus:** Regression and classification fundamentals

---

### 🔹 Assignment 4: Naive Bayes Classifier
- Probabilistic classification
- Gaussian and Multinomial Naive Bayes
- Conditional independence assumption
- Performance comparison with other classifiers

> **Focus:** Probabilistic learning models

---

### 🔹 Assignment 5: Supervised Learning Algorithms
- k-Nearest Neighbors (k-NN)
- Decision Trees
- Perceptron algorithm
- Support Vector Machines (SVM)
- Ridge regression and regularization

> **Focus:** Classical supervised learning techniques

---

### 🔹 Assignment 6: Neural Networks and Optimization
- Single-layer and multi-layer perceptrons
- Activation functions
- Loss functions
- Gradient descent and backpropagation
- Effect of learning rate and epochs

> **Focus:** Neural networks and optimization techniques

---

### 🔹 Assignment 7: Unsupervised Learning
- k-Means clustering
- Expectation-Maximization (EM) algorithm
- Gaussian Mixture Models (GMM)
- Principal Component Analysis (PCA)

> **Focus:** Structure discovery in unlabeled data

---

### 🔹 Assignment 8: Advanced ML Concepts
- Feature selection and dimensionality reduction
- Ensemble methods (Bagging, Boosting, Random Forests)
- Bias–variance tradeoff analysis
- Introduction to reinforcement learning concepts

> **Focus:** Advanced and applied Machine Learning methods
